Use case assessment checklist

Evaluate whether your use case meets the three fundamental requirements for synthetic health data generation

Critical requirement

Each of these three tests must be met in order for your synthetic data request to proceed to the next steps.

Types of synthetic data requests

This assessment applies to any of the following scenarios:

  • One organisation requests another organisation to generate a synthetic dataset for a specific project
  • An organisation wishes to establish a synthetic dataset for multiple potential projects or purposes
  • An organisation (or End User) requests to access or use a synthetic dataset that was created for a different purpose or use case
